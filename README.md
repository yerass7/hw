М2 – Basic Design Principles

## Description
В данном проекте представлены примеры корректной реализации
принципов проектирования DRY, KISS и YAGNI на языке C#.

## Principles Used

### DRY (Don't Repeat Yourself)
- Использование параметризованных методов
- Вынесение общих конфигураций

### KISS (Keep It Simple)
- Упрощение логики условий
- Отказ от ненужного LINQ
- Минимальное использование исключений

### YAGNI (You Aren’t Gonna Need It)
- Удаление лишних методов и настроек
- Реализация только необходимого функционала

## Technologies
- C#
- Object-Oriented Programming
